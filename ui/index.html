Welcome do BIPES!

							        

								    


									    
BIPES: Block based Integrated Platform for Embedded Systems allows text and block based programming for several types of embedded systems and Internet of Things modules using MicroPython, CircuitPython or Python. You can connect, program, debug a monitor several types of boards using network or USB. Please check a list of compatible boards here. Compatible boards include STM32, ESP32, ESP8266 and Raspberry Pi Pico.

									    
BIPES is fully open source and based on HTML and JavaScript, so no software install or configuration is needed and you can use it offline! We hope BIPES is useful for you and that you can enjoy using BIPES. If you need help, we now have a discussion forum. Feel free to use it! Also, if we invite you to use the forum to leave feedbacks and suggestions for BIPES!

									    
Thank you from the BIPES Team!
